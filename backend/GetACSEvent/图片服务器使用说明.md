# 门禁事件图片服务器使用说明

## 功能概述

本系统已集成图片服务器功能，可以将门禁事件相关的图片保存到本地，并通过Web URL进行访问。

## 主要功能

### 1. 图片自动保存
- 当门禁事件发生时，系统会自动保存相关图片到 `D:/Picture/{设备名称}/{员工姓名}/` 目录
- 图片文件名格式：`{员工姓名}_{年}{月}{日}_{时}{分}{秒}.jpg`
- 保存图片时会在控制台输出图片的本地路径和Web访问URL

### 2. Web图片访问
- 启动Web服务器后，可以通过以下方式访问图片：
  - 图片列表页面：`http://localhost:8080/images`
  - 直接访问图片：`http://localhost:8080/images/{设备名称}/{员工姓名}/{文件名}`

### 3. 事件API
- 事件列表API：`http://localhost:8080/events`
- 返回JSON格式的事件数据，包含：
  - 时间信息（设备原始时间，格式：yyyy-MM-dd HH:mm:ss）
  - 设备信息（IP、名称、ID、区域等）
  - 人员信息（员工号、姓名、卡号等）
  - 图片URL（自动替换localhost为本机IP）

### 4. 图片管理功能
- 图片列表页面提供以下功能：
  - 网格布局显示所有图片缩略图
  - 显示图片文件名、大小、修改时间
  - 点击图片可查看大图
  - 点击URL可复制图片访问地址
  - 按时间倒序排列，最新图片在前

## 配置说明

### Web服务器配置
在 `app.config` 文件中可以配置以下参数：

```xml
<appSettings>
  <!-- Web服务器配置 -->
  <add key="WebServerUrl" value="http://localhost:8080" />
  <add key="WebServerPort" value="8080" />
  <!-- 图片存储配置 -->
  <add key="PictureBasePath" value="D:/Picture" />
</appSettings>
```

### 配置参数说明
- `WebServerUrl`: Web服务器的基础URL地址
- `WebServerPort`: Web服务器监听端口
- `PictureBasePath`: 图片存储的基础路径

## 使用方法

### 1. 启动服务
运行程序后，Web服务器会自动启动，监听配置的端口。

### 2. 访问图片列表
在浏览器中打开：`http://localhost:8080/images`

### 3. 直接访问图片
图片URL格式：`http://localhost:8080/images/{设备名称}/{员工姓名}/{文件名}`

例如：
```
http://localhost:8080/images/前门门禁/张三/张三_20240115_143025.jpg
```

## 支持的图片格式

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- BMP (.bmp)
- WebP (.webp)

## 安全特性

- 路径安全检查：防止访问系统其他目录
- 文件类型验证：只允许访问图片文件
- 错误处理：友好的错误提示

## 注意事项

1. 确保 `D:/Picture` 目录存在且有写入权限
2. 确保Web服务器端口（默认8080）未被其他程序占用
3. 图片文件会按设备名称和员工姓名分类存储
4. 员工姓名中的特殊字符会被替换为下划线，确保文件夹名称合法
5. 建议定期清理旧图片文件以节省存储空间

## 故障排除

### 图片无法访问
1. 检查Web服务器是否正常启动
2. 确认图片文件是否存在
3. 检查文件路径是否正确

### 图片列表页面显示异常
1. 检查图片目录是否存在
2. 确认目录权限设置
3. 查看控制台错误信息

### 端口冲突
如果8080端口被占用，可以修改 `app.config` 中的 `WebServerPort` 配置。

### 图片命名问题
- 如果员工姓名包含特殊字符，系统会自动替换为下划线
- 如果设备名称为空，系统会使用设备IP作为设备名称
- 如果员工姓名为空，系统会使用"未知员工"作为文件夹名称

### 中文路径访问
- 系统支持中文设备名称和员工姓名的路径访问
- URL中的中文字符会自动进行编码和解码处理
- 支持直接使用中文路径和URL编码后的路径两种方式访问
- 示例：
  - 中文路径：`http://localhost:8080/images/前门门禁/张三/张三_20240115_143025.jpg`
  - 编码路径：`http://localhost:8080/images/%E5%89%8D%E9%97%A8%E9%97%A8%E7%A6%81/%E5%BC%A0%E4%B8%89/%E5%BC%A0%E4%B8%89_20240115_143025.jpg`
